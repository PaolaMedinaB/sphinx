Funcionalidades de la aplicación
================================

Tú aquí puedes hacer de todo

Formulario creación de "Descripción de cargo"
---------------------------------------------
Formulario para la descripción de un nuevo cargo

Formulario concepto sobre cargo
-------------------------------
Formulario para el registro del concepto sobre la solicitud de creación o actualización de cargo.

Ayudas para el diligenciamiento
-------------------------------
Ayudas contextuales para el diligenciamiento de los campos.

Formulario para diligenciamiento inicial sin proceso
----------------------------------------------------
Formulario para facilitar el cargue de la línea base de cargos con base en los archivos de excel.
