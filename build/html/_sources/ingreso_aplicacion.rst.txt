Ingreso a la aplicación
=======================

Requisitos del navegador
-----------------------
CA¿argos requiere conexión a internet y un navegador compatible con JavaScript, es decir, se puede usar con los siguientes navegadores: 

Microsoft Internet Explorer 7.0 o una versión posterior

Mozilla Firefox 14 o una versión posterior

Google Chrome 21 o una versión posterior

Apple Safari 5.0 o una versión posterior

Ingreso a la página de acceso
-----------------------------
Al ingresar  a la dirección http://dirección_ip:puerto/apex/f?p=número_del_aplicativo, aparecerá  la pantalla inicial de acceso al aplicativo. En esta ventana inicial se le solicitarán los campos para el ingreso de información de usuario y contraseña (que han sido asignados previamente):
